# 游戏机制更新说明 v1.1

## 📋 更新概述

根据新的产品需求，对游戏机制进行了以下调整：

### 变更日期：2024-12-15

---

## 🔄 主要变更

### 1. 广告观看奖励调整

**旧规则**：
- 每观看一次广告：+20积分
- 无每日上限

**新规则**：
- 每观看一次完整广告：+1积分
- 每日上限：20次（UTC+00:00重置）
- 达到上限后，广告仍可观看但不再获得积分

**影响**：
- 降低单次奖励，增加参与频率
- 防止刷广告行为
- 每日最多可通过广告获得20积分

### 2. 邀请奖励机制调整

**旧规则**：
- 邀请1人：立即获得+6积分
- 邀请满5人：额外+20积分（一次性）
- 邀请满10人：额外+30积分（一次性）

**新规则**：
- 邀请1人（且好友完成5次广告观看）：+6积分
- 邀请满10人（且好友完成5次广告观看）：额外+30积分（**可多次领取**）
- 取消5人里程碑奖励

**影响**：
- 强调好友质量而非数量
- 被邀请人需完成5次广告观看才触发奖励
- 邀请20人、30人等也会继续获得里程碑奖励

### 3. 签到奖励机制调整

**旧规则**：
- 连续签到1天：+4积分
- 连续签到3天：+15积分
- 连续签到7天：+30积分
- 连续签到15天：+30积分
- 连续签到30天：+60积分

**新规则**：
- **每日签到**：固定+4积分（每天都可领取）
- **连续签到7天**：额外+15积分（**一次性奖励**）
- **连续签到15天**：额外+30积分（**一次性奖励**）
- **连续签到30天**：额外+60积分（**一次性奖励**）
- 取消3天签到档位

**影响**：
- 每日固定4积分，奖励更稳定
- 里程碑奖励变为一次性，防止重复领取
- 中断签到后，从第1天重新开始计算

### 4. 每日签到加成保持不变

- 签到后激活 **1.36倍** 挖矿速度
- 持续时间：**2小时**
- 每日只能激活一次

### 每日最大积分获取

**旧规则**：
- 签到：4-60积分（看连续天数）
- 广告：无限制（理论上可以无限刷）
- 邀请：看运气
- **理论最大值**：无上限

**新规则**：
- 签到：4积分 + 里程碑奖励（首次达成时）
- 广告：最多20积分
- 邀请：看好友活跃度
- **每日稳定值**：24积分（签到4 + 广告20）

## 🛠️ 技术实现变更

### 代码文件修改

1. **backend/src/services/adService.js**
   - 添加 `DAILY_AD_LIMIT = 20`
   - 修改 `AD_REWARD_POINTS = 1`
   - 添加每日上限检查逻辑
   - 添加 `MIN_AD_WATCH_FOR_INVITE = 5`
   - 在第5次广告观看时触发邀请奖励

2. **backend/src/services/invitationRewardService.js**
   - 修改 `handleNewReferral()` 为不立即发放奖励
   - 新增 `processInvitationReward()` 方法
   - 修改10人里程碑为可多次领取
   - 移除5人里程碑奖励

3. **backend/src/services/checkInService.js**
   - 修改为固定4积分 + 里程碑额外奖励
   - 添加里程碑一次性领取检查
   - 更新 `MILESTONE_REWARDS` 配置

### 数据库迁移

**新迁移脚本**：`scripts/migrate/006_update_game_mechanics_for_new_rules.sql`

变更内容：
1. `user_check_in` 表新增字段：
   - `milestone_type` - 里程碑类型
   - `milestone_claimed` - 是否已领取

2. 更新 `check_in_reward_config` 配置
3. 禁用3天签到档位

## 📊 预期数据变化

### 用户积分增长速度

**旧规则下**（每日最大值）：
- 理论无上限（刷广告）

**新规则下**（每日最大值）：
- 稳定：24积分/天

- 邀请：需要好友活跃才发放
