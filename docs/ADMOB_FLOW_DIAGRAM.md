```
┌─────────────────────────────────────────────────────────────┐
│                       App启动 (main.dart)                      │
│  ┌──────────────────────────────────────────────────────┐   │
│  │  void main() async {                                  │   │

│  │  }                                                    │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                            ↓
```

## 🎬 广告加载与显示流程

```
┌───────────────┐
│   _loadAd()   │ 用户进入页面，自动触发
└───────┬───────┘
        │
        ↓
┌───────────────────────────────────────┐
│   setState(() {                        │
│     _isLoadingAd = true;  ← UI显示加载 │
│     _adLoadError = null;               │
│   });                                  │
└───────┬───────────────────────────────┘
        │
        ↓
┌───────────────────────────────────────┐
│  AdMobService.loadRewardedAd()        │ 请求AdMob服务器
│  ↓                                     │
│  根据debug模式选择:                     │
│  • Debug → 测试广告ID                  │
│  • Release → 生产广告ID                │
└───────┬───────────────────────────────┘
        │
        ├─────────────┬─────────────┐
        ↓             ↓             ↓
    成功 ✅         失败 ❌      超时 ⏱️
        │             │             │
        ↓             ↓             ↓
┌──────────┐   ┌──────────┐   ┌──────────┐
│ onAdLoaded│   │onAdFailed│   │  重试机制 │
└─────┬────┘   └────┬─────┘   └────┬─────┘
      │             │               │
      ↓             ↓               ↓
┌──────────┐   ┌──────────┐   ┌──────────┐
│_isAdReady│   │显示错误   │   │重新加载  │
│= true    │   │信息      │   │         │
│按钮启用  │   │按钮禁用  │   │         │
└─────┬────┘   └──────────┘   └──────────┘
      │
      ↓
┌─────────────────────────────────────────┐
│       UI状态显示（3种状态）               │
│  ┌───────────────────────────────────┐  │
│  │ 🔄 "广告加载中..." (黄色圆圈)      │  │
│  │    _isLoadingAd = true            │  │
│  ├───────────────────────────────────┤  │
│  │ ✅ "广告已准备好" (绿色对勾)       │  │
│  │    _isAdReady = true              │  │
│  ├───────────────────────────────────┤  │
│  │ ⚠️ "广告加载失败，请重试" (橙色)   │  │
│  │    _adLoadError != null           │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

## 👆 用户点击"观看广告"按钮

```
┌─────────────────────────────────────┐
│   用户点击按钮                        │
│   onPressed: () { ... }              │
└───────┬─────────────────────────────┘
        │
        ↓
┌─────────────────────────────────────┐
│   检查状态                            │
│   if (_isProcessing || !_isAdReady) │ ← 按钮禁用条件
│     return; // 按钮禁用              │
└───────┬─────────────────────────────┘
        │ 状态正常 ✅
        ↓
┌─────────────────────────────────────┐
│   _startWatchingAd()                │ 开始观看流程
└───────┬─────────────────────────────┘
        │
        ↓
┌─────────────────────────────────────┐
│   setState(() {                      │
│     _isWatchingAd = true; ← UI切换   │
│   });                                │
└───────┬─────────────────────────────┘
        │
        ↓
┌─────────────────────────────────────┐
│   final earnedReward =              │ 显示广告
│     await AdMobService              │
│       .showRewardedAd();            │
└───────┬─────────────────────────────┘
        │
        ├──────────────┬──────────────┐
        ↓              ↓              ↓
    看完 ✅         关闭 ❌        失败 ⚠️
  earnedReward   earnedReward    加载失败
    = true         = false          重新加载
        │              │              │
        ↓              ↓              ↓
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│_completeAd   │ │显示提示:      │ │返回初始状态  │
│Reward()      │ │"请观看完整   │ │重新加载广告  │
│↓            │ │广告"         │ │             │
│发放奖励     │ └──────────────┘ └──────────────┘
└──────┬───────┘
       │
       ↓
┌──────────────────────────────────────┐
│  调用后端API发放奖励                   │
│  ┌────────────────────────────────┐  │
│  │ if (isDailyCheckIn)             │  │
│  │   POST /check-in/daily          │  │
│  │   → 7.5Gh/s, 2小时              │  │
│  │ else                            │  │
│  │   POST /api/ad/extend-contract  │  │
│  │   → 5.5Gh/s, 2小时              │  │
│  └────────────────────────────────┘  │
└──────┬───────────────────────────────┘
       │
       ↓
┌──────────────────────────────────────┐
│  返回首页，传递成功状态                 │
│  Navigator.pop(context, true);        │
└──────────────────────────────────────┘
```

## 🔄 广告生命周期管理

```
┌─────────────────────────────────────────────────────────┐
│                    AdMobService                          │
│  ┌───────────────────────────────────────────────────┐  │
│  │  Singleton实例                                     │  │
│  │  _rewardedAd: RewardedAd? ← 当前广告对象          │  │
│  │  _isAdLoaded: bool ← 是否已加载                   │  │
│  │  _isAdLoading: bool ← 是否正在加载                │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  广告加载 (loadRewardedAd):                             │
│  ┌───────────────────────────────────────────────────┐  │
│  │  1. 检查是否正在加载/已加载                        │  │
│  │  2. 设置_isAdLoading = true                       │  │
│  │  3. 调用RewardedAd.load()                         │  │
│  │  4. 设置回调:                                      │  │
│  │     • onAdLoaded → _isAdLoaded = true            │  │
│  │     • onAdFailedToLoad → 触发错误回调            │  │
│  │     • onAdDismissedFullScreenContent → 重新加载  │  │
│  └───────────────────────────────────────────────────┘  │
│                                                          │
│  显示广告 (showRewardedAd):                             │
│  ┌───────────────────────────────────────────────────┐  │
│  │  1. 检查_rewardedAd是否为null                     │  │
│  │  2. 设置fullScreenContentCallback                 │  │
│  │  3. 调用_rewardedAd.show()                        │  │
│  │  4. 等待用户操作:                                  │  │
│  │     • 观看完整 → return true                      │  │
│  │     • 提前关闭 → return false                     │  │
│  │  5. dispose()释放资源                              │  │
│  │  6. 自动重新加载下一个广告                         │  │
│  └───────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────┘
```

## 💰 收益统计流程

```
用户观看广告 → AdMob服务器记录 → 收益累计
     ↓                ↓              ↓
  应用发放奖励    展示次数+1    预估收益增加
     ↓                ↓              ↓
  用户获得算力   填充率统计    24小时后可提现
```

## 📊 关键指标说明

```
┌────────────────────────────────────────────────────────┐
│  广告请求 (Ad Request)                                  │
│  ↓ 每次调用loadRewardedAd()计一次请求                  │
│                                                         │
│  广告填充 (Ad Filled)                                   │
│  ↓ AdMob服务器返回广告，填充率 = 填充/请求 × 100%       │
│                                                         │
│  广告展示 (Ad Impression)                               │
│  ↓ 用户看到广告，计费点                                │
│                                                         │
│  完整观看 (Completed View)                              │
│  ↓ 用户观看完整广告，获得奖励                           │
│                                                         │
│  预估收益 (Estimated Earnings)                          │
│  ↓ CPM × 展示次数 ÷ 1000                               │
└────────────────────────────────────────────────────────┘
```

## 🚦 状态管理总览

```
页面状态 (_AdRewardScreenState):
├── _isLoadingAd: bool ← 广告是否正在加载
├── _isAdReady: bool ← 广告是否准备好
├── _isWatchingAd: bool ← 用户是否正在观看广告
├── _isProcessing: bool ← 是否正在处理奖励
├── _adLoadError: String? ← 广告加载错误信息
└── _adMobService: AdMobService ← AdMob服务实例

按钮启用条件:
✅ 启用: _isAdReady == true && !_isProcessing
❌ 禁用: _isLoadingAd || !_isAdReady || _isProcessing

UI显示逻辑:
if (_isLoadingAd) 
  → 显示"广告加载中..."
else if (_isAdReady) 
  → 显示"广告已准备好"
else 
  → 显示"广告加载失败"
```
