   - 选择 "Add Database"
   - 数据库名: `bitcoin_mining_master`
   - 字符集: `utf8mb4`
   - 排序: `utf8mb4_unicode_ci`

3. **修改配置文件**
   
   编辑 `backend/.env`:
   ```env
   DB_HOST=localhost
   DB_USER=root
   DB_PASSWORD=你的MySQL密码
   DB_NAME=bitcoin_mining_master
   
   REDIS_HOST=localhost
   REDIS_PORT=6379
   REDIS_PASSWORD=
   ```

4. **运行数据库迁移**
   ```bash
   cd backend
   npx sequelize-cli db:migrate
   ```

5. **重启服务**
   ```bash
   pm2 restart bitcoin-backend
   ```

## 🆘 遇到问题?

### Redis 连接失败
```bash
# 检查状态
redis-cli ping

# 如果失败,重启
redis-cli shutdown
redis-server --daemonize yes
```

### MySQL 无法连接
```bash
# 检查进程
ps aux | grep mysqld

# 查看日志
tail -f /usr/local/mysql/data/*.err
```

### Sequel Ace 连接被拒绝
- 检查 MySQL 是否运行
- 确认用户名密码正确
- 尝试使用 Socket 连接方式
